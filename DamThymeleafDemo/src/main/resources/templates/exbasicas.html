Ejemplos de uso de expresiones

Expresiones literales

  • Número literal: número
  • Texto literal: texto
  • Booleano literal: booleano

Expresiones variables

  • Expresión variable (${today}): variable 1
  • Expresión variable (${nombre}): variable 2
  • Expresión variable (${producto.nombre}): variable 3
  • Expresión variable (${producto['descripcion']}): variable 4
  • Expresión variable (${lista[1].precio}): variable 5
  • Expresión variable (${lista.size()}): variable 6
  • Expresión variable (${map.p4.nombre}): variable 7

Expresiones de selección

  • Nombre: (*{nombre})nombre
  • Descripción (*{descripcion}): descripcion
  • Precio (*{precio}): precio

Expresiones de utilidad

Solo incluimos unos pocos ejemplos de la ingente cantidad que hay disponible
  • Expresión de utilidad (${#strings.toUpperCase(nombre)}) utilidad 1
  • Expresión de utilidad (${#strings.prepend(nombre,'D. ')}) utilidad 2
  • Expresión de utilidad (${#numbers.formatDecimal(producto.precio,3,2,'POINT')}) utilidad 3 €
  • Expresión de utilidad (${#numbers.formatCurrency(producto.precio)}) utilidad 3
  • Expresión de utilidad (${#numbers.formatPercent(0.01, 1, 0)}) utilidad 3
